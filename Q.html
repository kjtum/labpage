質問の際の注意

質問の際は，

(i) 誰が，(ii) どの状況で，(iii) 何をしようとしたときに，(iv) 何がおきたのか

を具体的に かつ 事象を切り分けて 書いた上で質問すること．

  1. 誰が: 質問者の学籍番号，名前
  2. どの状況で:
    ex) CLEでの履修した授業の「コンテンツ」に入ろうとした， その授業のひとつの「課題」「テスト」において， Windows PCにおいて， Cygwin を install したPCにおいて， マイクロソフトワードでレポートを作成しようとした， マイクロソフトワードで数式エディタを使用した
  3. 何をしようとしたのか：
    ex) Cygwin を Windowsマシンにinstallしようとした， C言語のコンパイル，「課題」を提出， 「テスト」に解答しようとした， ワードファイルをPDFに変換しようとした．
  4. 何がおきたのか，何が分からないのか:
    ex) install時にどのようなエラーが出たのか， どのようなコンパイルエラーが具体的に出たのか， ファイル添付の方法が分からない， 数式の入力方法がわからない

e-mail送信の際の注意

  1. メールサブジェクト（件名）を分かりやすく書くこと（科目名と質問である旨を記載）
